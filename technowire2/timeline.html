Programming languages have evolved, comprising machine code for computers, assembly languages for simplified programming, high-level languages like FORTRAN, 4GL for specific problem-solving such as SQL, and object-oriented languages like Java. Understanding these categories is vital in comprehending the fundamentals of programming languages and their significance in computer science.

1957 Fortran

Fortran is a general-purpose, compiled imperative programming language that is especially suited to numeric computation and scientific computing.

1959 Cobol

COBOL (Common Business-Oriented Language) is a compiled English-like computer programming language designed for business use.

1964 Basic

BASIC (Beginners' All-purpose Symbolic Instruction Code) is a family of general-purpose, high-level programming languages.

1969 C

C is a general-purpose, procedural computer programming language supporting structured programming, lexical variable scope, and recursion.

1970 Pascal

Pascal is an imperative and procedural programming language, designed by Niklaus Wirth as a small, efficient language intended to encourage good programming practices using structured programming and data structuring.

1983 C++

C++ is a general-purpose programming language created as an extension of the C programming language, providing object-oriented features and generic programming support.

1987 Perl

Perl is a family of two high-level, general-purpose, interpreted, dynamic programming languages, Perl 5 and Perl 6.

1991 Python

Python is an interpreted, high-level, general-purpose programming language known for its readability and clear syntax.

1993 Ruby

Ruby is an interpreted, high-level, general-purpose programming language known for its simplicity and productivity.

1995 Php

PHP is a popular general-purpose scripting language that is especially suited to web development.

1995 Java

Java is a general-purpose, class-based, object-oriented programming language designed to have as few implementation dependencies as possible.

1995 Javascript

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification.

2005 Ruby on rails

Ruby on Rails, often referred to as Rails, is an open-source server-side web application framework written in Ruby.
